About Commune

Hi! My name is Jeff Eng and I'm the developer of Commune.

This web app was inspired by a final project challenge for the Harvard CS50 Intro to Computer Science course. My motivation for this app was to solve a practical problem where I would loan things out to friends and family, then end up forgetting who borrowed which item. This app aims to solve that problem by creating an inventory system with ability to check out and return items like you would in a library.

The app is built using Django and Django REST framework integrated with JavaScript to provide a seamless user experience and minimize page reloads.

Thank you for making a rest stop at Commune on your road trip around the web!
